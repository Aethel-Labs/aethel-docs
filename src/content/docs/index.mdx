## Getting Started

Make sure you have [Node.JS](https://nodejs.org/en) and [Bun](https://bun.com) installed in your device.

### Cloning & Installing

```sh
git clone https://github.com/aethel-labs/aethel.git
cd aethel
bun i 
cd web
bun i
```
