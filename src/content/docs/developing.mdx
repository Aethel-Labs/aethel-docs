## Getting started

Thank you for your interest in contributing to Aethel, before you start, make sure you read our [Countributing Guide](https://github.com/Aethel-Labs/aethel?tab=contributing-ov-file) and [Code of Conduct](https://github.com/Aethel-Labs/aethel?tab=coc-ov-file)


### Cloning the repository

To develop for Aethel, fork the repository and switch to the `dev` branch

```sh
git clone https://github.com/aethel-labs/aethel.git
git switch dev
```

Now, you are free to add new changes, features, and bug fixes to Aethel, when you are done, open a pull request.

Make sure you format, build and lint to make sure Aethel is clean and free of errors

```sh
bun run format
bun run lint
bun run build
```
